The Technologies behind Android Care

Android Care

Android Care was created using a range of different technologies to cater for the three different subsystems:

  • The smart watch as worn by the patient
  • The cloud database that processes all events generated by the smart watch
  • The web application dashboard that allows carers to log in and monitor patient's treatment

The Smart Watch SubSystem

Android Wear

An extension of Android, Android Wear is designed for wearable devices, namely smart watches.

Android

The application is currently being developed on the Android platform using Android Studio

Java

Of course, Java is the underlying language of Android.

The Cloud SubSystem

ElasticSearch

ElasticSearch is a lucene based search engine, a NoSQL database, storing JSON objects

Kibana

Kibana is used as a tool to visualise objects stored in ElasticSearch

The Carer's SubSystem | Web Application

Python

The carer's dashboard was developed using Python due to its simplicity in working with a MySql database

MySql

MySql is utilised by Python to store patient's and carer's credentials

Bootstrap

Finally, the Twitter Bootstrap CSS framework is used and overridden to create a custom Bootstrap template

HTML5

The web application needs HTML to display content via a web browser

CSS3

This content is wrapped and 'made pretty' using CSS3

AngularJS

Angular provides a more fluid UX, injecting content rather than loading when content is requested by the browser

JQuery

JQuery is required as an underlying language for Angular and Bootstrap JS
